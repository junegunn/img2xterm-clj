## Usage

```
Usage: java -jar img2xterm-*.jar [OPTIONS] IMAGE...
Options:
  -w, --width WIDTH    Max width (default: terminal width)
  -h, --height HEIGHT  Max height (default: terminal height)
      --help           Show this message
```
